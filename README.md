Ejercicio 1: Blog personal

### 🎯 Objetivo

Crear una maqueta web funcional para un **blog personal**, utilizando HTML5 estructurado, estilos en CSS3 y un toque de interactividad con JavaScript.

### 🧱 Requisitos mínimos

- Uso de etiquetas semánticas (`header`, `main`, `section`, `article`, `footer`, etc.).
- Diseño responsivo usando Flexbox o Grid.
- Estructura separada en archivos: `index.html`, `estilos.css` y `script.js`.
- Incorporar algún tipo de interactividad con JS (alertas, validaciones, sliders, menús, etc.).

### 🧩 Componentes sugeridos

- Lista de artículos
- Sidebar con categorías
- Footer con enlaces
